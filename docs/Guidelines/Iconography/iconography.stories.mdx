# Iconography

`import { Alert } from '@ticketswap/solar/icons'`

## Usage

`<Alert title="Alert" size={24} />`

### Props

| name  | type   | description      | default           |
| ----- | ------ | ---------------- | ----------------- |
| size  | number | Icon size        | 32                |
| color | string | Icon color       | space             |
| title | string | aria-label title | current icon name |
